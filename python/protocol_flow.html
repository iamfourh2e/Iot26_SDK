READdriver polls FC03/FC04 → multi-register → int decode → scale
WRITEFC06 write_register or FC10 write_registers via downlink command
PORT502 default, simulator uses 5020
Modbus RTU (RS-485)
READSerial frame FC03/FC04 → CRC check → registers → scale
WRITEFC06 via serial — write_register(reg, val, slave)
BUS/dev/ttyUSB0, 9600 8N1, up to 32 slaves
I²C
READwrite_i2c_block_data(cmd) → wait read_delay_ms → read N bytes → MSB decode
WRITEi2c_write command (config registers, reset, forced mode)
BUS/dev/i2c-1, up to 128 addresses on one bus
SPI
READxfer2(cmd bytes) → clock in response → bit_order decode → raw float
WRITExfer2(write command) — configuration / mode register writes
PINMOSI/MISO/CLK/CS, up to 1 device per CE pin
Analog ADC (MCP3208)
READSPI xfer2 → 12-bit raw count → scale_factor + offset → engineering value
WRITEN/A — ADC is read-only; write to DAC separately if needed
USE4–20 mA, NTC thermistor, potentiometer, LDR
1-Wire (DS18B20)
READcat /sys/bus/w1/devices/{rom}/temperature → millidegrees → scale 0.001
WRITEN/A — DS18B20 is read-only (temp only)
BUSGPIO4, single wire, up to ~20 probes, kernel driver
